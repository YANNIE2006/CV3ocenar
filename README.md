Project Overview
A fun, interactive CV/Resume generator that allows users to input their information, upload a photo, and instantly generate a styled CV. Built with PHP form handling using `$_POST` and `$_FILES` superglobals.

Features
- User Information Form - Collects name, email, phone, career summary, education, and skills
- Photo Upload - Users can upload their profile picture using `$_FILES` superglobal
- Dynamic CV Generation - Displays the submitted information in a professional CV format
- Animated Background - Gradient background with smooth color transitions
- Floating Sticker Animations - Fun, floating emojis that add personality
- Responsive Design - Works on desktop and mobile devices

  Technologies Used
| Technology | Purpose |
|------------|---------|
| PHP | Backend form handling with `$_POST` and `$_FILES` |
| HTML5 | Form structure and CV display |
| CSS3 | Styling, animations, gradients, and responsive design |
| XAMPP | Local development environment |
